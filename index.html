API Plan

This is a prototype for my subway map API training project, providing a framework for introducing different concepts along the API life cycle.

My original subway map API for this project was a hypermedia API (Siren), and this next version is a static (Jekyll) implemention of the same model, but using Jekyll hosted on Github as the client.

This is all a work in progress, and there is no visual aspects of this implementation yet, but you can find the functionality for one of the API areas (design) here/a>, with the Siren collection on Github, and the templates that act as the client.
